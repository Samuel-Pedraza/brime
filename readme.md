# Intro

This is a project framework, still in the works, that is going to support IE8.

### About

Syntax: How to make a single row, add a full width column, and add padding to an element.

<code>
<div class="row">
  <div class="col-1>
  <p class="padding-small">Hello World</p>
  </div>
  </div>
</code>
